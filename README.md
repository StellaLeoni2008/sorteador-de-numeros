## 📌 Sobre

O *Sorteador de Números* foi desenvolvido como parte das atividades propostas no curso **Alura – Lógica de Programação: praticando com desafios**.
O curso teve como foco principal a prática de lógica de programação com JavaScript, sem aprofundamento em HTML e CSS. O objetivo foi desenvolver o raciocínio lógico por meio da resolução de desafios e da construção de funcionalidades dinâmicas utilizando JavaScript.

Neste repositório, apresento o código do projeto e a aplicação prática dos conceitos trabalhados ao longo do curso.

## 🧠 Conceitos Fundamentais Praticados

- Durante o desenvolvimento deste projeto, foram aplicados conceitos essenciais de programação, como:
- Estruturação e organização de variáveis
- Conversão e validação de dados de entrada
- Uso de condicionais para controle de fluxo
- Estruturas de repetição
- Manipulação de arrays
- Criação e organização de funções
- Geração de números aleatórios
- Manipulação do DOM para atualização dinâmica da interface
- Separação de responsabilidades no código

Este projeto foi fundamental para fortalecer minha base em JavaScript, consolidar minha lógica de programação e compreender como a linguagem interage com a interface da aplicação.
